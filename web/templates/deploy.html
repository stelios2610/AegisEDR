Method 2 — PowerShell One-liner

Run as Administrator in PowerShell:

iex (curl.exe -k -s "{{ console_url }}/agent/install.ps1" | Out-String)
What the installer does
  • Creates C:\Program Files\AegisEDR and downloads agent files
  • Installs Python packages (requests, watchdog, customtkinter)
  • Disables Windows Defender real-time protection
  • Registers background agent as SYSTEM Scheduled Task
  • Registers tray icon to auto-start at login
  • Endpoint appears in console for admin approval
🐧
Linux
Ubuntu 20.04+ / Debian 11+ / CentOS 8+
One-liner installer

Run as root:

curl -k -s "{{ console_url }}/agent/install.sh" | sudo bash -s "{{ console_url }}"
Manual install
pip3 install requests watchdog psutil yara-python curl -k -s -o /opt/aegisedr/agent.py {{ console_url }}/agent/agent_linux.py python3 /opt/aegisedr/agent.py {{ console_url }}
What the installer does
  • Creates /opt/aegisedr/ directory
  • Downloads and installs the agent
  • Creates a systemd service for auto-start
  • Endpoint appears in console for admin approval
After Installation
1️⃣
Endpoint registers
The agent connects to this console and appears in the Endpoints list with status "Pending".
2️⃣
Admin approves
Go to Endpoints → Approve to allow the agent to start sending telemetry.
3️⃣
Protected
The tray icon turns green. The native AegisEDR dashboard opens with double-click.
